# Quick Deploy (GitHub Pages)

1) Put *both* files in the **same folder** (root of repo or /docs):
   - index.html
   - map.html
2) Settings → Pages → Deploy from a branch:
   - If you used root: Branch = main, Folder = /root
   - If you used /docs: Branch = main, Folder = /docs
3) Open the Pages URL shown by GitHub.

If the map 404s:
- Confirm map.html is in the same folder as index.html.
- The filename must be exactly `map.html` (case sensitive).
- Clear cache (Ctrl+F5) or try incognito.
